Como usar

Escolha um modo, cole suas entradas e clique no botão principal. O objetivo deste conceito é deixar o Sortick mais parecido com uma ferramenta direta do que com uma página promocional.

Sobre este conceito

Esta é uma reconstrução visual alternativa para testes no repositório de testes. A ideia é reduzir texto, priorizar botões e deixar os modos mais próximos de uma ferramenta aberta.
