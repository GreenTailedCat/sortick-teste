BINGO

Sorteie números de bingo com histórico visível

O modo Bingo foi pensado para acompanhar cartelas físicas ou uma atividade de grupo. O primeiro número usa contagem de início; os próximos seguem em animação curta para não atrasar o ritmo da partida.

EXEMPLOS

Quando este modo ajuda

Ideias práticas

  • Bingo em confraternização.
  • Atividade de sala com cartelas físicas.
  • Sorteio de números em evento pequeno.
  • Dinâmica em comunidade ou equipe.

Um limite importante

Sem repetição, cada número sai uma vez. Com repetição ativada, o mesmo número pode voltar a aparecer.

Os dados ficam salvos no navegador usado para criar o sorteio. Para levar a mesma lista a outro dispositivo, use o resumo, uma imagem ou recrie a lista no outro navegador.
