SORTEIO POR NOMES

Sorteie nomes, opções ou uma ordem de forma clara

Adicione uma lista e escolha entre um resultado, vários sorteados ou uma ordem completa. É um formato simples para situações em que um grupo precisa de uma escolha aleatória sem perder a organização.

PASSO A PASSO

Como usar este modo

1

Adicione uma lista

Digite um nome por vez ou cole vários nomes de uma só vez.

2

Escolha o formato

Selecione um resultado, vários sorteados ou uma ordem completa.

3

Defina a rodada

Ative a opção de não repetir para evitar que o mesmo nome apareça de novo.

4

Compartilhe

Copie ou baixe o resultado depois do sorteio.

EXEMPLOS

Quando este modo ajuda

Ideias práticas

  • Escolher quem começa uma atividade.
  • Definir ordem de apresentações ou falas.
  • Sortear opções em um grupo.
  • Selecionar participantes de uma dinâmica.

Um limite importante

Quando a opção de não repetir está ativa, use “Reiniciar rodada” para liberar novamente todos os participantes.

Os dados ficam salvos no navegador usado para criar o sorteio. Para levar a mesma lista a outro dispositivo, use o resumo, uma imagem ou recrie a lista no outro navegador.
