COMO FUNCIONA

Crie, organize e decida direto no navegador.

Escolha um modo, configure a atividade, adicione participantes ou números quando necessário e use o resultado. Os dados ficam salvos neste navegador para você continuar depois.

1. Configure

Informe nome, descrição, data e o formato que faz sentido para a atividade.

2. Use o modo

Sorteie nomes, gire a roleta, monte grupos, acompanhe uma cartela, jogue dados ou faça um bingo.

3. Registre

Copie, compartilhe ou baixe o resultado quando essa opção estiver disponível.

Criar atividade

Sortick é uma ferramenta de organização e transparência. Não vende bilhetes, não processa pagamentos e não administra premiações.
