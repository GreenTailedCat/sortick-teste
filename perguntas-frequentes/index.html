Direto ao ponto

O essencial antes de usar.

Preciso criar uma conta?

Não. Você pode criar e usar sorteios direto no navegador.

Onde meus sorteios ficam salvos?

No navegador e dispositivo usados para criar o sorteio. Eles não são sincronizados automaticamente entre aparelhos.

Funciona no celular?

Sim. O site foi pensado para funcionar em celular e computador, respeitando o espaço disponível na tela.

Posso continuar uma atividade depois?

Sim, enquanto os dados do navegador continuarem lá. A página inicial mostra os sorteios salvos nesse navegador.

Posso usar a mesma lista em outro modo?

Nos modos compatíveis, é possível duplicar ou reutilizar a lista para criar outro sorteio sem digitar os nomes novamente.

O Sortick vende números, recebe pagamentos ou administra prêmios?

Não. A ferramenta organiza listas, números e resultados. O organizador é responsável pelo uso e pelas regras aplicáveis.

O resultado é compartilhável?

Dependendo do modo, você pode copiar o resumo, usar o compartilhamento do navegador ou baixar uma imagem do resultado.

Encontrei um erro ou tenho uma ideia.

Use o link de Erro/sugestões? no rodapé para enviar uma mensagem com o modo utilizado e uma descrição do que aconteceu.

Sortick é uma ferramenta de organização e transparência. Não vende bilhetes, não processa pagamentos e não administra premiações.
