ROLETA ONLINE

Use uma roleta quando o momento pede algo mais visual

A roleta usa a mesma lógica do sorteio por nomes, mas apresenta a escolha em uma animação circular. Ela é útil quando o resultado faz parte da atividade, não apenas do registro final.

PASSO A PASSO

Como usar este modo

1

Insira nomes ou opções

A roleta pode representar pessoas, temas, tarefas ou qualquer lista curta.

2

Escolha o formato

Gire para obter um resultado, vários resultados ou uma ordem completa.

3

Controle repetições

Ative não repetir quando cada item deve sair apenas uma vez na rodada.

4

Mostre o resultado

O giro visual torna a escolha fácil de acompanhar pelo grupo.

EXEMPLOS

Quando este modo ajuda

Ideias práticas

  • Escolher um tema para começar.
  • Definir uma pessoa para responder.
  • Selecionar uma tarefa do dia.
  • Criar uma dinâmica de perguntas ou desafios.

Um limite importante

Para listas muito grandes, os rótulos da roleta ficam resumidos para manter a leitura visual.

Os dados ficam salvos no navegador usado para criar o sorteio. Para levar a mesma lista a outro dispositivo, use o resumo, uma imagem ou recrie a lista no outro navegador.
