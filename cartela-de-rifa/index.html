NÚMERO DA SORTE

Organize uma cartela de números em um único lugar

A cartela serve para organizar números e associá-los a nomes. Ela permite acompanhar o que ainda está disponível, o que foi ocupado e o que está confirmado, sem processar pagamentos ou vender números.

PASSO A PASSO

Como usar este modo

1

Monte a cartela

Defina o título e a quantidade de números, de 2 a 500.

2

Adicione detalhes

Prêmio, imagem, valor, data e observação são opcionais.

3

Marque cada número

Toque em um número disponível para associar um nome; toque no ocupado para alternar o status.

4

Exporte quando precisar

Gere uma imagem completa para conferir ou compartilhar o estado da cartela.

EXEMPLOS

Quando este modo ajuda

Ideias práticas

  • Organizar números de uma rifa familiar.
  • Controlar reservas de uma atividade.
  • Acompanhar uma lista numerada de participantes.
  • Compartilhar uma cartela atualizada em imagem.

Um limite importante

O Sortick organiza a cartela; o responsável pela atividade define regras, pagamentos, prêmios e obrigações aplicáveis.

Os dados ficam salvos no navegador usado para criar o sorteio. Para levar a mesma lista a outro dispositivo, use o resumo, uma imagem ou recrie a lista no outro navegador.
