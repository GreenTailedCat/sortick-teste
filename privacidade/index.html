Privacidade

Política de Privacidade

Versão inicial para a fase de testes do Sortick. Antes de um lançamento público com cadastro, anúncios ou pagamentos, esta página deve ser revisada.

1. O que o Sortick coleta nesta versão

Nesta versão offline/local, os sorteios criados ficam salvos no próprio navegador do usuário, usando armazenamento local. Isso pode incluir nome do sorteio, participantes, números associados, status de confirmação e resultado.

2. Onde os dados ficam

Os dados ficam no dispositivo e navegador em que o sorteio foi criado. Se o usuário limpar os dados do navegador, trocar de dispositivo ou abrir em outro navegador, esses sorteios podem não aparecer.

3. Dados de participantes

O organizador deve evitar inserir dados sensíveis ou desnecessários. Para o uso comum, recomenda-se usar apenas nomes simples, apelidos ou identificações que façam sentido para o grupo.

4. Compartilhamento

Quando o usuário usa os botões de copiar, compartilhar ou baixar imagem, ele escolhe onde enviar o resultado. O Sortick não controla os aplicativos, redes sociais ou serviços externos usados para esse compartilhamento.

5. Analytics, anúncios e login

O Sortick pode usar Google Analytics para medir acessos e Google AdSense para anúncios. Os dados dos sorteios criados pelo usuário, como nomes de participantes, números associados, status e resultados, continuam armazenados localmente no navegador e não são enviados pelo Sortick para esses serviços.

6. Segurança

Como esta versão funciona localmente, a segurança também depende do dispositivo do usuário. Evite usar o Sortick para guardar informações confidenciais.

7. Contato

Esta é uma página inicial de privacidade para desenvolvimento. Informações definitivas de contato devem ser adicionadas antes da publicação oficial.

Google AdSense e anúncios

O Sortick pode exibir anúncios por meio do Google AdSense. O Google pode usar cookies, identificadores e tecnologias semelhantes para exibir, limitar e medir anúncios.

O Sortick não envia ao Google AdSense os nomes dos participantes, números associados a pessoas ou conteúdo específico criado dentro dos sorteios.

Sortick é uma ferramenta de organização e transparência. Não vende bilhetes, não processa pagamentos e não administra premiações.
