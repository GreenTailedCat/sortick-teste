GRUPOS E TIMES

Divida participantes em grupos de forma equilibrada

O modo Grupos/Times embaralha a lista e reparte os participantes de forma equilibrada. Você pode deixar os nomes padrão ou personalizar cada grupo antes de gerar a divisão.

EXEMPLOS

Quando este modo ajuda

Ideias práticas

  • Montar grupos de trabalho.
  • Dividir times para jogo.
  • Separar participantes de uma oficina.
  • Organizar equipes de uma dinâmica.

Um limite importante

A divisão busca manter quantidades próximas entre os grupos; ela não considera habilidades, preferências ou critérios externos.

Os dados ficam salvos no navegador usado para criar o sorteio. Para levar a mesma lista a outro dispositivo, use o resumo, uma imagem ou recrie a lista no outro navegador.
